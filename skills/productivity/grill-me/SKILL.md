# Grill Me

Run a `grilling` session.

## When to Use

Use when the user wants to be grilled, stress-test a plan, sharpen a design, prepare for review, choose among tradeoffs, or convert vague intent into a clear goal and next action.

Use `scrutinize` when the user wants findings on an already written plan, PR, or design doc. Use an implementation skill when the user asks for direct changes instead of an interview.
